# Zx_RieOS v1.1.1

**Zx_RieOS: A Computational Framework for Riemann-Hilbert Correspondence in Cosmology**

DOI: 10.5281/zenodo.19981688

## Data
- `Zx_RieOS_v1.1.csv`: Main dataset with 38,000 entries
- `Zx_Island_2119.csv`: Island universe parameters  
- `Zx_Periodic_Hierarchy.csv`: Periodic hierarchy results
- `Zx_Planck_Epochs.csv`: Planck epoch analysis
- `Zx_Spacetime_Wells.csv`: Spacetime wells simulation

## Code
- `main.ipynb`: Run this to reproduce all figures. Uses `Zx_RieOS_v1.1.csv`.
